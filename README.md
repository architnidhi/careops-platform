# 🏥 CareOps Platform

A unified operations platform for healthcare service businesses, built for the CareOps Hackathon.

## 🎯 Problem Statement
Healthcare service businesses struggle with fragmented operations - patient scheduling, care team coordination, and protocol management often exist in separate systems.

## 💡 Solution
CareOps Platform integrates:
- **Patient Management**: Centralized patient records and history
- **Care Flow Automation**: Digital protocols for common procedures
- **Team Coordination**: Real-time task assignment and tracking
- **Analytics Dashboard**: Key metrics for operational efficiency

## 🚀 Features
- ✅ Patient intake and management
- ✅ Real-time backend API
- ✅ Modern React frontend
- ⬜ Custom care protocol builder
- ⬜ Reporting dashboard

## 🛠️ Tech Stack
- **Backend**: Python/Flask
- **Frontend**: React.js
- **Database**: PostgreSQL (coming soon)
- **Authentication**: JWT (coming soon)

## 📋 Prerequisites
- Python 3.9+
- Node.js 14+
- Git

## ⚙️ Installation

### Backend Setup
\\\ash
cd backend
python -m venv venv
# On Windows: venv\Scripts\activate
# On Mac/Linux: source venv/bin/activate
pip install -r requirements.txt
python app.py
\\\

### Frontend Setup
\\\ash
cd frontend
npm install
npm start
\\\

## 🌐 Usage
1. Start backend: \http://localhost:5000\
2. Start frontend: \http://localhost:3000\
3. Add patients using the form
4. View patient list in real-time
